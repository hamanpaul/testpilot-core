# TestPilot Core Issues #4 and #18 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 讓 `testpilot --version` 顯示 core 與所有 entry-point plugins 的版本組合，並把現有 remediation coordinator 接回 domain-agnostic 的 tier-1-first / tier-2 one-shot env-recovery 架構。

**Architecture:** #18 直接沿用 `testpilot.plugins` entry-point metadata，逐 plugin fail-soft 讀 distribution version 與 class `api_version`。#4 保留 plugin-owned deterministic tier-1，新增 core-owned prompt/schema/one-shot Copilot adapter；tier-1 連續兩次未修復才在 retry 間隙升 tier-2，agent 只產生 plugin capability catalog 允許的 env plan，plugin 執行後 core 強制呼叫 deterministic `verify_env`，並把完整 audit 與 `agent-recovered` marker 寫入 case trace/run payload。

**Tech Stack:** Python 3.11+、Click、`importlib.metadata`、github-copilot-sdk 0.1.x dynamic adapter、pytest、JSON run artifacts。

## Global Constraints

- 不修改或操作 serialwrap runtime、daemon、session、DUT、STA；所有驗證只使用 core unit/integration fakes。
- core 不得 import 或命名任何 wifi/domain-specific module、device model 或 command。
- agent 只能在 `on_retry` env-recovery phase 產生 plan，不能接觸 test steps、pass criteria 或 verdict。
- tier-2 之後一定由 core 呼叫 plugin `verify_env`；LLM 自稱成功不能放行。
- 每個 task 完成後立即 commit，不跨 task 累積未提交 diff。

```python
def _plugin_version_lines() -> list[str]:
    try:
        entry_points = list(importlib.metadata.entry_points(group=PluginLoader.ENTRY_POINT_GROUP))
    except Exception:
        return []
    lines: list[str] = []
    for entry_point in sorted(entry_points, key=lambda item: str(item.name)):
        name = str(entry_point.name)
        dist = getattr(entry_point, "dist", None)
        dist_name = str(getattr(dist, "name", "") or name)
        try:
            plugin_version = importlib.metadata.version(dist_name)
        except Exception:
            plugin_version = "unknown"
        try:
            plugin_class = entry_point.load()
            api_version = str(getattr(plugin_class, "api_version", "") or "unknown")
        except Exception:
            api_version = "unknown"
        lines.append(f"  plugin {name} {plugin_version} (api {api_version})")
    return lines
```

```python
def send_one_shot(self, request, prompt, *, timeout_seconds=60.0) -> str:
    async def _op(client):
        session = await client.create_session(request.to_create_config(permission_handler))
        try:
            event = await session.send_and_wait({"prompt": prompt}, timeout=timeout_seconds)
            content = str(getattr(getattr(event, "data", None), "content", "") or "").strip()
            if not content:
                raise CopilotSDKUnavailableError("Copilot one-shot returned no assistant content")
            return content
        except asyncio.TimeoutError:
            await session.abort()
            raise
        finally:
            await client.delete_session(session.session_id)
    return self._run(self._with_client(_op))
```
